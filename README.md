# Pixel Luxe Portfolio

A modern, minimal, and premium portfolio website inspired by high-end fashion design aesthetics. Built using HTML, CSS, and JavaScript with a focus on clean UI, smooth animations, and responsiveness.

## ✨ Features

* Minimal and luxury-inspired design
* Fully responsive (mobile + desktop)
* Smooth scrolling and animations
* Grid-based project showcase
* Interactive UI elements
* Clean typography and spacing

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```
pixel-luxe-portfolio/
│── index.html
│── style.css
│── script.js
│── assets/
│   ├── images/
│   └── icons/
```

## 🚀 Getting Started

1. Clone the repository

```
git clone https://github.com/vaibhavkhator/pixel-luxe-portfolio.git
```

2. Open the project folder
3. Run `index.html` in your browser
